## 📌 Overview
This project demonstrates the use of:
- List Comprehension
- Dictionary Comprehension
- Generators
- Traditional (nested loop) approach

## 🧠 Concepts Covered

### ✅ List Comprehension
- Compact way to create lists
- Faster and more readable

### ✅ Dictionary Comprehension
- Efficient key-value mapping
- Reduces code complexity

### ✅ Generators
- Memory efficient
- Generates values one-by-one using `yield` and generator expressions
