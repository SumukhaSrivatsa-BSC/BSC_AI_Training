# AI Learning Assignment (Python + OOP)
Overview
This repository contains solutions to a complete AI learning assignment covering core Python concepts and Object-Oriented Programming (OOP).
The goal of this assignment is to build a strong foundation in Python while applying it to simple AI-related scenarios like spam detection.

Topics Covered-
Python Basics
User input and output
Conditional statements
Loops and functions

Functions-
Creating reusable functions
Input-based logic (e.g., spam detection, accuracy calculation)

Object-Oriented Programming (OOP)-
Classes and Objects
Constructors (__init__)
Methods
Encapsulation
Inheritance
Polymorphism
Abstraction

File Handling-
Reading and writing files
Appending data to files
Handling persistent storage

Exception Handling-
Handling invalid input
Managing runtime errors (e.g., division by zero, file not found)

Logging-
Recording application activity using Python logging module
Creating log files for tracking system behavior
