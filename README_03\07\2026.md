# Machine Learning for AI - Employee Performance Prediction

## Overview
This project applies **Machine Learning, Feature Engineering, Data Analysis, and Model Evaluation** on a corporate employee dataset to predict **Employee Performance Score**.

## Dataset Information
Dataset: **Corporate_AI_Employee_Dataset.csv**

Main features include:

- Employee ID
- Department
- Gender
- Education Level
- Job Role
- Experience Years
- Salary
- Coding Test Score
- Communication Score
- AI Training Hours
- Promotion History
- Attrition Risk
- Employee Performance Score (Target Variable)

## Project Tasks

### Data Understanding
- Dataset exploration and summary.
- Identification of numerical and categorical features.
- Memory optimization and data quality checks.
- Missing value and duplicate analysis.

### Exploratory Data Analysis
- Correlation matrix and multicollinearity analysis (VIF).
- Skewness detection.
- Department-wise and education-wise performance analysis.
- Visualization using Matplotlib and Seaborn.

### Feature Engineering
- Experience Level creation.
- Salary Band categorization.
- Performance Index calculation.
- Interaction feature generation.
- One-Hot Encoding of categorical variables.

### Feature Selection
- Mutual Information
- Recursive Feature Elimination (RFE)
- SelectKBest
- Common important feature identification

### Machine Learning Models
The following regression models were trained and compared:

- Linear Regression
- Ridge Regression
- Lasso Regression
- ElasticNet
- Decision Tree Regressor
- Random Forest Regressor
- Extra Trees Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

### Model Evaluation
Models were evaluated using:

- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)
- R² Score
- Adjusted R² Score
- Cross Validation

### Hyperparameter Tuning
- GridSearchCV (Random Forest)
- RandomizedSearchCV (XGBoost)

### Model Interpretation
- SHAP Feature Importance
- Actual vs Predicted Analysis
- Residual Analysis

### Deployment
- Pipeline creation using Scikit-learn.
- Model saved as:

`employee_performance_model.pkl`

## Technologies Used
- Python
- Pandas

- Scikit-learn
- XGBoost

## Outcome
The project identifies key factors affecting employee performance and builds optimized machine learning models to accurately predict employee performance scores.
