# NLP Basics Assignment

This repository contains solutions for the **NLP Basics Assignment** from Cambridge Infotech.

##  Topics Covered

- Introduction to Natural Language Processing (NLP)
- Tokenization
- Stopwords
- Text Lowercasing
- Rule-Based NLP Systems
- Basic Python String Operations
- Pandas DataFrames
- Text Processing using Pandas
- Sentiment Analysis Dataset Examples

##  Technologies Used

- Python 3
- Pandas
- Google Colab

##  Assignment Structure

### Section A – Theoretical Questions
Theoretical concepts related to NLP including:

- What is NLP?
- NLP Applications
- NLP Pipeline
- Tokenization
- Stopwords
- Rule-Based Systems
- Pandas Basics for NLP

### Section B – Practical Questions
Python and Pandas programs covering:

- String manipulation
- Character counting
- Word counting
- Spam detection
- DataFrame creation
- DataFrame filtering
- Sentiment analysis datasets
- Text preprocessing tasks

##  How to Run

### Option 1: Google Colab

1. Open Google Colab.
2. Create a new notebook.
3. Copy and paste the code from the assignment.
4. Run each cell individually.

### Option 2: Jupyter Notebook

```bash
pip install pandas
```

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook and execute all cells.
